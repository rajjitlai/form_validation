# Modern Form Validation

A sleek, premium form validation application featuring a **Black & Dark Glassmorphism** aesthetic. This project demonstrates modern UI/UX principles with real-time feedback and smooth animations.

## ✨ Features

- 🌑 **Premium Dark Theme**: Deep blacks and dark grays for a modern, high-end feel.
- 🪟 **Glassmorphism UI**: Semi-transparent card with high-blur background and subtle borders.
- ⚡ **Real-time Validation**: Immediate feedback as you type, improving user experience.
- 🎨 **Modern Typography**: Uses the **Outfit** variable font for a clean, contemporary look.
- 📱 **Responsive Design**: Fully optimized for various screen sizes.
- 🧼 **Clean Code**: Refactored logic for better maintainability and performance.

## 🚀 Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (Variables), Glassmorphism, and smooth transitions.
- **JavaScript (ES6+)**: Functional validation logic with event-driven updates.
- **FontAwesome**: High-quality icons for visual feedback.

## 🛠️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/rajjitlai/form_validation.git
   ```
2. **Open the project**:
   Simply open `index.html` in your preferred browser.
